# v1.0.0 - Initial Release

## Why

Initial release of zylos-telegram component for Telegram Bot integration.

## What

### Added Files
- `src/bot.js` - Main bot logic using Telegraf
- `src/send.js` - Send interface for C4 protocol
- `src/admin.js` - Admin CLI for configuration
- `src/lib/auth.js` - Authorization helpers
- `src/lib/config.js` - Configuration management
- `src/lib/media.js` - Media download handling
- `install.js` - Installation script
- `upgrade.js` - Upgrade script
- `uninstall.js` - Uninstall script
- `ecosystem.config.js` - PM2 configuration

### Configuration
- Config stored at: `~/zylos/components/telegram/config.json`
- Media stored at: `~/zylos/components/telegram/media/`
- Logs at: `~/zylos/components/telegram/logs/`

### Environment Variables
- `TELEGRAM_BOT_TOKEN` - Bot token from @BotFather (required)
- `TELEGRAM_PROXY_URL` - Optional proxy URL

## How

### Fresh Install
```bash
zylos-pm install telegram
```

### Manual Setup
1. Clone repo to `~/.claude/skills/telegram`
2. Run `node install.js`
3. Add `TELEGRAM_BOT_TOKEN` to `~/zylos/.env`
4. Restart: `pm2 restart zylos-telegram`
